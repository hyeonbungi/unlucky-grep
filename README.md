# Unlucky Grep

언러키 그렙(Unlucky Grep)은 커맨드 라인 인터페이스에서 사용할 수 있는 유사 Grep 프로그램입니다. 특정 파일에서 개행 문자로 구분된 패턴을 검색하여 각 행을 출력합니다.

## Specifications

* 작성 언어: Rust
* 구동 플랫폼: Windows, Linux, macOS

## Usages

```Bash
> unlucky-grep [search_query] [file_path]
```
